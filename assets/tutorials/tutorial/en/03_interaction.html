Make a right-click anywhere on the ground to switch to interaction mode. The cursor should look like this:

When the interaction mode is enabled, interactive objects will be highlighted in a yellowish glow.
To open the interaction menu, make a left-click on it after you've enabled interaction mode.

Depending on the target, the following interactions may appear:
The use interaction is the default interaction with objects: picking up items, opening doors, using a computer, etc.
The inspect interaction may give you additional details about an object. Higher perception might give you access to more insights.
Use skill allows you to attempt to use a specific skill on an object. Some use cases could be: using lockpick on a door, using steal on a character, or medicine on yourself.
Use spell works in the same way as Use skill, but for your magic spells (if you have any).
Use item uses an item on a target. Some use cases are: unlocking a door with a matching key, or using a health potion.
Talk to will attempt to initiate a conversation with the target.
Shove can be used on some NPCs. If one of them is blocking your path, before pulling out the big guns and blowing them away, you might also try this interaction to politely ask them to move away.
