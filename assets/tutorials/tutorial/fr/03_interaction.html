Faites un clic avec le bouton droit de la souris, n'importe où sur le sol, pour passer en mode interaction. Le pointeur devrait maintenant ressembler à cela:

Lorsque vous êtes en mode interaction, les objects avec lesquels vous pouvez intéragir sont mis en évidence grâce à une halo jaune.
Pour ouvrir le menu d'interaction, faites un clic sur un objet avec le bouton gauche de la souris.

Selon l'objet sur lequel vous avez cliqué, les interactions suivantes vous seront suggérées:
Utiliser est l'interaction par défaut: elle ouvre les portes, ramasse les objets qui peuvent l'être, fouille les placards, etc.
L'inspection peut vous permettre d'obtenir des détails supplémentaires sur un objet. Les personnages avec une haute perception peuvent obtenir une description plus détaillée encore.
Utiliser une compétence vous permet de mettre à profit une de vos compétences. Certains cas d'utilisations peuvent être: crochetage sur une porte, vol sur un personnage, médecine sur un blessé.
Utiliser un sort fonctionne de la même façon que Utiliser une compétence, pour l'utilisation de votre magie (si vous en avez).
Utiliser objet vous permet d'utiliser un objet depuis votre inventaire sur la cible. Certains cas d'utilisations peuvent être: déverouiller une porte avec sa clef, ou utiliser une potion de vie.
Parler vous permet d'essayer d'entamer une conversation avec un personnage.
Pousser peut être utilisé sur certains personnages non-joueurs. Si l'un d'eux vous bloque un chemin, avant de sortir vos flingues et d'en faire un tas de cendre, considérez la possibilité de poliement lui demander de dégager le chemin.

Et maintenant ?
Essayez d'utiliser l'interaction Utiliser sur la porte, pour tenter d'ouvrir la porte.
