Félicitations ! Vous avez atteint la fin de ce tutorial.

Devant vous, vous devriez voir que le sol à une teinte marron. Cela vous indique la présence d'une zone de sortie. Marcher dans une zone de sortie vous téléportera ailleurs.

Il y a deux types de zones de sortie :
Les zones de sortie locales vous téléporteront dans un autre niveau, ou ailleurs dans le même niveau.
Les zones de sorties globales vous renverront vers la carte du monde.

Avancez vers le sol marron pour atteindre la Carte du monde.
